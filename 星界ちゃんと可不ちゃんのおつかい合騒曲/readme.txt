・楽曲データを入れて遊ぶ方法

1.楽曲データを用意してください。ファイル形式は「.ogg」です。
2.譜面データファイル(曲名.tja)と同じ場所に楽曲データを配置してください。
3.tjaファイルをメモ帳等で開き、「WAVE:」の箇所に先ほど用意した楽曲データのファイル名を記述してください
(例...「WAVE:さいたま2000.ogg」)

以上で楽曲データの導入は可能です。以下、必要があれば調節してください

ex1.音符のタイミングにズレを感じる場合は、tjaファイル内の「OFFSET:」で楽曲の再生タイミングを調節してください。
(例...「OFFSET:-0.5」と記述すると、譜面が0.5秒遅れて流れてきます)
ex2.楽曲の音量が大きいor小さいと感じる場合は、tjaファイル内の「SONGVOL:」の値を調節してください。
ex3.「DEMOSTART:」の値を調節することにより、曲選択画面での楽曲の再生タイミングを調節可能です。
